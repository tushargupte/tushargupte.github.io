    • 7—IBM Intranet
    Navigation

Client—IBM

Agency—VSA Partners/IBM Studio

Problem

Over the years, IBM Intranet navigation had grown to include many sections and sub-sections that are not always relevant to the users. To redesign and simplify the navigation of IBM's global intranet, internally called w3.

Solution

The recommended solution was to tailor content that is appropriate to the type of user and personalize the user experience accordingly. The navigation interface would be unobtrusive, simple and easy to use on both desktop as well as mobile devices.
